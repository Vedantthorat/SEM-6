1. Which of the following is a basic unit of an Artificial Neural Network (ANN)?

A) Neuron
B) Cluster
C) Kernel
D) Module

Answer: A) Neuron

2. What activation function is most commonly used in hidden layers of deep networks due to its computational efficiency and gradient propagation?

A) Sigmoid
B) Tanh
C) ReLU
D) Softmax

Answer: C) ReLU

3. In a Multilayer Perceptron (MLP), what type of learning is typically used?

A) Reinforcement learning
B) Unsupervised learning
C) Supervised learning
D) Self-supervised learning

Answer: C) Supervised learning

4. What is the main purpose of the forward propagation step in an ANN?

A) To adjust weights
B) To compute loss
C) To predict output based on input
D) To calculate gradients

Answer: C) To predict output based on input

5. What is the goal of back propagation in neural networks?

A) Generate test data
B) Optimize learning rate
C) Minimize error by updating weights
D) Normalize the dataset

Answer: C) Minimize error by updating weights

6. Which of the following is NOT a building block of a deep neural network?

A) Activation function
B) Loss function
C) Recurrent loop
D) Weight parameters

Answer: C) Recurrent loop

7. Gradient Descent works by:

A) Increasing the loss at each iteration
B) Finding the global maxima
C) Minimizing the loss function
D) Randomly selecting weights

Answer: C) Minimizing the loss function

8. Which of the following is not a variant of gradient descent?

A) Batch Gradient Descent
B) Stochastic Gradient Descent
C) Mini-batch Gradient Descent
D) Gaussian Gradient Descent

Answer: D) Gaussian Gradient Descent

9. What does the derivative of the loss function represent in gradient descent?

A) Size of the network
B) Learning rate
C) Slope or rate of change of the error
D) Final accuracy

Answer: C) Slope or rate of change of the error

10. Batch Optimization refers to:

A) Training one sample at a time
B) Updating weights after entire dataset is processed
C) Ignoring gradients
D) Using very large learning rates

Answer: B) Updating weights after entire dataset is processed

11. What does the Momentum Optimizer help to overcome?

A) High accuracy
B) Slow convergence and local minima
C) Model underfitting
D) Overfitting

Answer: B) Slow convergence and local minima

12. RMSProp optimizer adapts learning rates based on:

A) Running average of gradients
B) Momentum
C) Learning rate schedule
D) Epoch size

Answer: A) Running average of gradients

13. In a Multilayer Perceptron (MLP), what is the role of hidden layers?

A) Process output
B) Extract features
C) Receive input data
D) Determine the number of neurons

Answer: B) Extract features

14. What is the purpose of cross-validation in ANN training?

A) To evaluate the generalization performance of the model
B) To split the data into training and test sets
C) To perform hyperparameter tuning
D) None of the above

Answer: A) To evaluate the generalization performance of the model

15. What is the primary difference between Linear and Logistic Regression in deep networks?

A) Number of layers
B) Output activation function
C) Training algorithm
D) Type of loss function used

Answer: B) Output activation function

16. Logistic Regression is most suited for:

A) Regression problems
B) Multi-label classification
C) Binary classification
D) Time series forecasting

Answer: C) Binary classification

17. Which loss function is most commonly used in linear regression?

A) Cross-Entropy Loss
B) Mean Absolute Error
C) Mean Squared Error
D) Hinge Loss

Answer: C) Mean Squared Error

18. Which of the following statements is false about choosing learning rate in gradient descent?

A) Small learning rate leads to slow convergence
B) Large learning rate cause the loss function to fluctuate around the minimum
C) Large learning rate can cause divergence
D) Small learning rate cause the training to progress very fast

Answer: D) Small learning rate cause the training to progress very fast

19. In backpropagation, the chain rule is used to compute:

A) Weight Initialization
B) Gradient Of Loss With Respect To Weights
C) Output Of The Network
D) Regularization Term

Answer: B) Gradient Of Loss With Respect To Weights

20. Output of sigmoid activation function ranges from:

A) 0 To 1
B) -1 To 1
C) -1 To 0
D) 0 To 9

Answer: A) 0 To 1
